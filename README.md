# 🛒 E-Commerce Website Frontend

A fully responsive, modern E-Commerce frontend built with **React** and **Tailwind CSS**.  
Powered by the FakeStore API for dynamic product data, with a focus on clean UI and smooth UX.

---

## 🚀 Features

| Feature | Description |
|---|---|
| 🔐 Authentication | Secure login/signup via Clerk |
| 🛍️ Product Listing | Dynamic products from FakeStore API |
| 📄 Product Details | Dedicated pages for each product |
| 🛒 Cart UI | Add to cart with smooth interactions |
| ⚡ Fast API calls | Axios-powered data fetching |
| 🎠 Carousels | Product sliders with Swiper & React Slick |
| 🔔 Notifications | Toast alerts via React Toastify |
| 📱 Responsive | Fully mobile-friendly design |
| ✨ Animations | Smooth Lottie animations |
| ⬆️ Scroll to Top | Quick navigation utility |

---

## 🛠️ Tech Stack

**Frontend:** React, React Router DOM  
**Styling:** Tailwind CSS v4, Tailwind Vite Plugin  
**API:** Axios, FakeStore API  
**Auth:** Clerk  
**UI/Animations:** Swiper.js, React Slick, Lottie React, Lucide React, React Icons, React Toastify  

---

## ⚡ Getting Started

```bash
# 1. Clone the repository
git clone https://github.com/nabeel-builds/E-commerce-Website-Frontend.git

# 2. Navigate into the project
cd E-commerce-Website-Frontend

# 3. Install dependencies
npm install

# 4. Start the dev server
npm run dev
```
